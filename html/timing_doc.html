Timer Configuratie

Fast interval timer (ACQ_TRIGGER):

  • k_timer periodic mode met 30s interval
  • Automatic restart (geen manual re-arm nodig)
  • Drift-free: kernel compenseert execution time

Slow interval timer (PUBLISH_REQUEST):

  • k_timer periodic mode met 600s (10 min) interval
  • Offset: -15s (initial start 585s, zodat publish 15s voor einde interval gebeurt)
  • Rationale offset: geeft modules tijd om laatste sample te nemen voor publish

Drift-Free Guarantee: Zephyr k_timer periodic mode compenseert voor callback execution time, geen cumulatieve drift over tijd.
