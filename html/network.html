Kort Overzicht

De Network module beheert de LTE-M/NB-IoT modem connectie en implementeert een state machine voor robuuste netwerk connectiviteit.

Functionaliteit

  • Modem initialisatie - Nordic nRF9161 modem configuratie
  • LTE-M / NB-IoT - Dual-mode support met automatische fallback
  • State machine - Connectie beheer met states: disconnected, connecting, connected
  • Network events - L4 (connectivity) event handling
  • PSM/eDRX - Power saving modes voor battery efficiency
  • ZBUS interface - Publiceert network status updates

ZBUS Channels

Subscribed:

  • REBOOT_CHAN - Graceful modem shutdown bij reboot

Published:

  • NETWORK_STATUS_CHAN - Network connectie status updates
