Kort Overzicht

De Cloud module beheert de connectie met nRF Cloud en implementeert een state machine voor robuuste cloud connectiviteit met retry logica.

Functionaliteit

  • nRF Cloud SDK integratie - Gebruikt Nordic's nRF Cloud library
  • State machine - Connectie beheer met states: disconnected, connecting, ready
  • Event handling - Reageert op cloud events (connected, disconnected, data_received)
  • Retry logica - Exponential backoff bij connectie failures
  • ZBUS interface - Publiceert cloud status updates

ZBUS Channels

Subscribed:

  • NETWORK_STATUS_CHAN - Wacht op network voordat cloud connectie start
  • REBOOT_CHAN - Graceful cloud disconnect bij shutdown

Published:

  • CLOUD_STATUS_CHAN - Cloud connectie status updates
