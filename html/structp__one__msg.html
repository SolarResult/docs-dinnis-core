Detailed Description

P_ONE module message structure.

Message containing P1 DSMR meter data updates. Published on P_ONE_CHAN when new data is available.

Uses int32_t for all energy/power values to avoid FPU usage and reduce memory. Energy in Watt-hours (Wh), Power in Watts (W). OBIS F9.3.3 format: 999999.999 kWh = 999999999 Wh fits in int32_t (2.1 billion max).

All timestamps are Unix epoch milliseconds from P1 meter (OBIS 0-0:1.0.0).
