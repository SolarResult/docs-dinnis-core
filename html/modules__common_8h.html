Macro to compute the maximum message size from a list of channel data types.

The macro expands to a list of sizeof(type) for each channel's type, followed by a 0 to account for the trailing comma when expanding SIZE_OF_TYPE. The MAX_N macro is used to find the maximum value in the list. Example: MAX_N(sizeof(struct cloud_msg), sizeof(enum fota_msg_type), 0)

Parameters
_CHAN_LISTList of channels to compute the maximum message size from. The list should be in the format: (CHANNEL_NAME, type)
Returns
Maximum message size from the list of channels
