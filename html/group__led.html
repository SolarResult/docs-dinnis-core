enum led_status_priority

LED status priorities.

Defines the priority levels for LED status indication. Higher numeric values indicate higher priority. When multiple statuses are active, the highest priority is displayed.

Note: LED module is a consumer-only module that determines its status internally based on network, cloud, and trigger channel messages. It does not publish any messages or provide a channel.
